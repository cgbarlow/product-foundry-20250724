# Ravi's Adventure

A hilarious text adventure game about debugging AI systems and the absurd situations that arise when code meets comedy.

## 🎮 About

Join Ravi, a developer who finds themselves trapped in a surreal world where debugging isn't just about fixing code—it's about surviving increasingly ridiculous scenarios involving malfunctioning AI systems, sentient algorithms, and the occasional existential crisis.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start the adventure
npm start

# Development mode (with file watching)
npm run dev
```

## 🧪 Development

```bash
# Run tests
npm test

# Run tests in watch mode
npm test:watch

# Check code quality
npm run lint

# Fix linting issues
npm run lint:fix
```

## 📁 Project Structure

```
ravis-adventure/
├── src/           # Game source code
├── data/          # Game data (rooms, items, characters)
├── saves/         # Player save files
├── tests/         # Test files
├── docs/          # Documentation
└── config/        # Configuration files
```
